Tips: To enter an accented letter (áéíóú) hold down 'AltGr' and press the letter. For other letters, hold down 'Alt' and type the number pad code:

  • for ü, type 0252
  • for ñ, type 0241
  • for ¡, type 0161
  • for ¿, type 0191
  • for «, type 0171
  • for », type 0187
